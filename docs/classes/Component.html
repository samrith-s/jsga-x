    Component Class

    Methods:

    Component class allows creation of new components

    Constructor

    Component

    (
    • name
    • target
    • [options]
    )
    Object

    Parameters:

    • name String
      • A unique name for the component
    • target Object
      • A screen or layer object to add the component to
    • [options] Object optional
      • A JSON object of options

    Returns:

    Object:

    Current object after creating

    Example:

    var scrn = new Screen('myScreen');
var layr = new Layer('myScreen', scrn);
var myComp = new Component('myComponent', layr, {
    type: 'img',
    classes: 'class1 class2',
    image: {
        path:'path/to/img.png',
        alt: 'alt Img Name',
        title: 'title of image'
    }
});

    changeParent

    (
    • current
    • target
    )
    Object

    Change the current parent container of the layer or component object

    Parameters:

    • current Object
      • The current parent (screen or layer) object
    • target Object
      • The target screen or layer object

    Returns:

    Object:

    Current object after creating

    Example:

    var scrn = new Screen('myScreen');
var scrn2 = new Screen('myOtherScreen');
var myComp = new Component('myComp', scrn, {
    type: 'img',
    classes: 'class1 class2',
    image: {
        path:'path/to/img.png',
        alt: 'alt Img Name',
        title: 'title of image'
    }
});
myComp.changeParent(scrn, scrn2);
