    • Methods

    Methods

    display

    (
    • target
    • options
    )
    Object chainable

    Display the timer in a specific layer or screen object

    Parameters:

    • target Object
      • Screen or layer object to embed the timer in
    • options Object
      • A JSON object of options
      • [iconPath] String optional

        Path to the icon for the timer

      • [showIcon] Bool optional

        Set whether to display the icon for the timer or not

      • [col] String optional

        The column (between 1-12) to specify the width of the HUD. Accepts 'auto', too

      • [row] String optional

        The row (1-12) to specify the height of the HUD. Accepts 'auto', too

      • [x] String optional

        The left position based on the grid (0 - 12).

      • [y] String optional

        The top position based on the grid (0 - 12).

    Returns:

    Object:

    Current object after updating

    Example:

    var scrn = new Screen('myScreen');
var timr = new Timer('myTimer', 10, true);
timr.display(scrn, {col:2, row:5, x:3, y:2});
