    Layer Class

    Methods:

    Layer class that allows creation of new layers

    Constructor

    Layer

    (
    • name
    • target
    • [options]
    )
    Object

    Parameters:

    • name String
      • A unique name for the layer
    • target Object
      • A screen or layer object to add the layer to
    • [options] Object optional
      • A JSON object of options
      • [col] String optional

        The column (between 1-12) to specify the width of the Layer. Accepts 'auto', too

      • [row] String optional

        The row (1-12) to specify the height of the HUD. Accepts 'auto', too

      • [x] String optional

        The left position based on the grid (0 - 12)

      • [y] String optional

        The top position based on the grid (0 - 12)

    Returns:

    Object:

    Current object after creating

    Example:

    var scrn = new Screen('myScreen');
var layr = new Layer('myScreen', scrn, {col:12, row: 'auto', x:0, y:0});
