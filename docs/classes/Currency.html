    Currency Class

    Methods:

    Currency class that allows creation of new currencies like health, coins etc and allows effective management. Part of the game logic element.

    • min
    • max
    • value

    )
    Object

    Parameters:

    • name String
      • A unique name for the currency
    • min Int
      • Minimum value of currency
    • max Int
      • Maximum value of currency
    • value Int
      • Value to initialize currency with
    • [options] Object optional
      • Pass a JSON object of options

    Returns:

    Object:

    Current object after creating

    Example:

    var health = new Currency('health', 0, 100, 100, {callback: myFunct(), icon:'../../myicon.png'});
