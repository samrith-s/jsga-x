    Menu Class

    Menu class that allows creation of new Menu with icons

    Constructor

    Menu

    (
    • name
    • screen
    • options
    )
    Object

    Parameters:

    • name String
      • A unique name for the Menu
    • screen Object
      • Name of the screen
    • options Object
      • Pass a JSON object of options
      • [col] String optional

        The column (between 1-12) to specify the width of the HUD. Accepts 'auto', too

      • [row] String optional

        The row (1-12) to specify the height of the HUD. Accepts 'auto', too

      • [x] String optional

        The left position based on the grid (0 - 12).

      • [y] String optional

        The top position based on the grid (0 - 12).

    Returns:

    Object:

    Current object after creating

    Example:

    var scrn = new Screen('myScreen');
var mnu = new Menu('mainMenu', scrn, {col: 8, row: 'auto', x:2, y:2});
